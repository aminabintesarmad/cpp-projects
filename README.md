This GitHub repository contains a simple C++ program designed for beginners. It helps new learners understand basic programming concepts, logic building, and simple game implementation in C++.

🎮 Project Included
Q1. Head and Tail Game

This program is a simple coin-toss style game where the user selects either "Head" or "Tail", and the program randomly generates a result. It helps beginners understand:

Basic input/output in C++
Conditional statements (if-else)
Random number generation
Simple game logic implementation

Q2. Random Number Generator Game

This program is a simple number-based game where the computer generates a random number, and the user tries to guess it. It helps beginners understand:

Basic input/output in C++
Conditional statements (if-else)
Random number generation
User interaction and guessing logic
Simple game development concepts
